# Alex Hiriavenko CV Website

Personal multilingual CV website of Alex Hiriavenko, Fullstack Developer specializing in Vue.js, Node.js, NestJS, Laravel, PostgreSQL, CRM systems, API development, and performance optimization.

Live site: https://alexhiriavenko.github.io/

## Tech Stack

- Gulp 5
- Nunjucks
- SCSS
- Webpack
- JavaScript
- i18n static routing

## Local Development

Install dependencies:

```bash
npm i
```

Run development server:

```bash
npm run dev
```

Build production files:

```bash
npm run product
```

## Deploy

Deploy to GitHub Pages:

```bash
npm run githubDeploy
```

The production site is published at:

```text
https://alexhiriavenko.github.io/
```
